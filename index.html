End-to-end automation systems I've built

Contract Automation System

Problem
Employment agency spent 10-15 hours weekly manually creating 8 contracts per client, re-typing data across documents.
Solution
Built Make.com workflow: Google Form intake → database lookup → 8 document generation in PandaDoc → e-signature routing → Google Drive organization → email monitoring for approvals.
Outcome
2 minutes per contract (down from 10-15 hours). 500-600 hours saved annually. Zero filing errors.
Make.com PandaDoc API Google Sheets Gmail

Receipts Report Automation

Problem
10+ hours weekly manually extracting date, amount, currency from hundreds of receipts for expense reporting.
Solution
Two-part Make.com system: watches new folders → creates "Logged" subfolder → copies template sheet → processes receipts with Gemini AI → extracts data → handles multi-currency with live exchange rates.
Outcome
5 minutes weekly (down from 10+ hours). 100% automated data extraction. Real-time currency conversion.
Make.com Google Gemini AI Google Drive API Exchange Rate API

AI Video Production Pipeline

Problem
2-3 hours per video for scriptwriting, voiceover recording, caption generation — bottlenecked to 2-3 videos/week.
Solution
n8n workflow: AI script generation (Llama 3.3) → text-to-speech conversion → GPT-4 caption timing → Google Drive asset management.
Outcome
5 minutes per video (down from 2+ hours). Scaled from 2-3 to 10+ videos/week capacity.
n8n OpenRouter API GPT-4 Llama 3.3

Lead Capture & Auto-Response

Problem
Website form submissions had no tracking, no immediate follow-up, manual data entry to spreadsheets.
Solution
Webhook-triggered Make.com workflow: captures submissions → logs to Google Sheets → instant email to owner → auto-acknowledgment to lead.
Outcome
100% lead capture. 5-second acknowledgment time. Zero manual data entry. Complete audit trail.
Make.com Webhooks Google Sheets API Gmail API

About Me

I'm Marc Lawrence Salaguban, a UX designer and automation builder with experience in operations, procurement, and product work.

I started by managing government workflows where accuracy mattered. I later moved into UX work across healthcare, finance, and education. I used automation to speed up research, communication, and admin tasks.

Now I help teams and organizations save time with intelligent automation systems. I build with Zapier, Make, n8n, Google Workspace, Notion, webhooks, and AI tools.
